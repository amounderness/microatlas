# MicroAtlas Closed Beta Plan

## Purpose

The closed beta is intended to test whether MicroAtlas works as a moderated public atlas and registry for micronations, self-declared states, and experimental civic projects.

## Beta objectives

- Test account creation and verification.
- Test nation creation, claim drawing, and flag upload.
- Test submission and moderation workflows.
- Test public atlas/profile visibility.
- Test reporting, hiding, restoration, and ban workflows.
- Collect feedback on clarity, trust, safety, and usability.

## Tester instructions

Testers should attempt to:

1. Create an account.
2. Complete their profile.
3. Create a nation draft.
4. Draw a claim.
5. Upload a flag.
6. Submit for review.
7. View approved entries on the atlas.
8. Report a test entry if instructed.
9. Send feedback on confusing or broken steps.

## Beta rules

- Testers should not submit personal addresses.
- Testers should not upload hateful, pornographic, illegal, or malicious content.
- Testers should understand that this is an MVP and data may be removed before public launch.
